伸缩侧边栏的案例，使用 CSS + JavaScript

# 配色

文本 / icon : rgba(255,255,255,.6)

侧边栏背景色：rgba(0,0,0,.8)

  • 鼠标滑过背景色：rgba(0,0,0,.1)

仿 IOS 关闭栏

  • 红#eb5a56
  • 黄#f8bc33
  • 绿#62cb44

宽度

  • 鼠标未滑入：110px
  • 鼠标滑入：280px

长度

  • 暂定为 560px

思路

  • 整体由 width 增加与 transtion 实现
  • 布局由 Flex 实现
  • 图片的语法：background: url(img/bg.jpg) center no-repeat; 居中显示与无重复显示
